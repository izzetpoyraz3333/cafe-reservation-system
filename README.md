☕ Cab Cafe. - Digital Cafe & Reservation Management System

A full-stack web application built for cafes and restaurants, allowing customers to place orders and managers to track them in real-time.
✨ Features

* **Real-time Order Tracking:** Powered by Firebase, customer orders appear on the admin dashboard instantly.
* **Order Status Management:** Admins can update orders as "Preparing", "Delivered", or "Cancelled".
* **Dine-in & Takeaway:** Easily filter orders based on delivery type.
* **Reservation System:** Customers can book tables, and admins can assign tables and approve requests dynamically.
* **Modern UI:** Built with Next.js and Tailwind CSS, fully responsive.

## 🛠️ Technologies Used

* **Frontend:** Next.js, React, TypeScript, Tailwind CSS
* **Backend / Database:** Firebase (Firestore)
* **State Management:** React Hooks

## 💻 Running Locally

To run this project on your local machine:

1. Clone the repo: `git clone https://github.com/izzetpoyraz3333/cab-reservation-system.git`
2. Navigate to the folder: `cd repo-name`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`
